## ğŸ” Understanding the Interface

### Left Panel: Code Editor
- **BASIC Tab**: Write high-level code
- **Assembly Tab**: View generated assembly (or write your own)

### Center: Circuit Visualization
- **CPU**: Contains the ALU and registers
- **RAM**: Memory storage
- **Buses**: Data and address pathways (animate during execution)

### Right Panel: I/O
- **Input**: Enter values for INPUT commands
- **Output**: See PRINT command results

### Bottom: Memory Panel
- View RAM contents and stack
